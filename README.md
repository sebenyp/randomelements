# randomelements

HU:
Ezt az oldalat egy egyetemi tárgyamhoz készítettem.
Like-old az oldal alján található gomb segítségével, hogy jó jegyet kapjak. Köszönöm!

A periódusos rendszer elemeit mutatja meg véletlenszerûen, ezt a funkciót le lehet állítani.
A periódusos tábla kinézetét megváltoztathatod az alsó gombok segítségével.
Az egérrel körbejárhatod a táblát, közelíteni illetve távolodni lehet a görgõvel.

Az elkészítéshez felhasznált eszközök megtalálhatók az oldal tetején található linkekre kattintva.
